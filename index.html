not so sketchy

Josi is a web framework for node.js.

Getting started

Create a josi app and start the dev server:

$ josi temp
$ cd temp
$ josi run

A josi controller:

var results = require('josi/results');
var redirect = results.redirect;
var view = results.view;

this.index = function() {
  return view({ title: 'A Title', name: 'Kate' });
};
this.redir = function() {
  return redirect('/product/details/1');
}
